Sequences are the core unit of work in OpenStory. Each sequence represents a video project that progresses from script to finished video.

## Creating a New Sequence

1. Navigate to **Sequences** from the top navigation
2. Click **New Sequence**
3. Enter your script or idea — you can paste a full screenplay or describe a concept
4. OpenStory will analyze your script and break it into scenes

## Understanding the Sequence Workflow

Each sequence follows a pipeline:

1. **Script** — Your original text, analyzed into individual scenes
2. **Scenes** — Each scene gets a visual prompt, camera direction, and mood treatment
3. **Cast** — Characters identified from the script with consistent visual references
4. **Locations** — Settings extracted from the script for visual consistency
5. **Theatre** — Preview and export your final video

## Working with Scenes

After script analysis, each scene appears as a card with:

- **Thumbnail** — Generated image representing the scene
- **Motion** — Image-to-video animation
- **Metadata** — Title, duration, location, time of day, story beat

You can regenerate any scene's image or motion independently.

## Tips

- Write descriptive scripts for better scene analysis
- Use the **Cast** tab to ensure character consistency across scenes
- Review scenes in the **Theatre** tab before exporting
